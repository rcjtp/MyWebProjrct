# Queuie
โปรเจคระบบวินมอเตอร์ไซค์รับ-ส่งลูกค้าผ่าน Application จัดทำโดย จตุพร เรือนฟ้า
### Features
- ปักหมุดสถานที่ที่ต้องการจะไป
- กดเรียกรถเพื่อให้วินมอเตอร์ไซค์เข้าไปรับ-ส่ง
- เลือกชำระเงินด้วยเงินสดหรือผ่าน Mobile Banking
